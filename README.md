# CallCenter Pro — Demo

Enterprise VoIP + AI call center platform demo (React + Vite).

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy

Works out of the box on [Vercel](https://vercel.com). Connect this GitHub repo — no extra config needed.

**Build command:** `npm run build`  
**Output directory:** `dist`

## Demo screens

| Route | Screen |
|-------|--------|
| `/` | Landing |
| `/agent` | Agent Desktop |
| `/supervisor` | Supervisor Wallboard |
| `/admin` | Admin / PBX Console |
| `/calls` | Call History & CRM |
| `/analytics` | Analytics |
